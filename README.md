# 亦仁视角 (yiren-shengcai-perspective)

> 从草根到生财有术创始人的思维框架

## 简介

本 Skill 提供亦仁的商业思维框架和赚钱方法论，涵盖套利思维、精益创业、流量获取等核心模型。

**适用场景：**
- 分析赚钱机会和商业项目
- 探索流量渠道和变现模式  
- 学习低成本验证和 MVP 思维
- 理解信息差、认知差、执行差的商业逻辑

## 效果示例

### 💡 用户提问：
> "有个做短视频的想法，你觉得能成吗？"

### 🎯 亦仁视角分析：

**1. 先验证需求是否真实**
- 去抖音/小红书搜索同类内容，看数据
- 直接问10个目标用户，愿不愿意付费/关注

**2. 算笔账**
- 获客成本：做一个视频要多久？能触达多少人？
- 变现路径：广告？带货？私域？客单价多少？
- 盈亏平衡点：多久能回本？

**3. MVP 验证**
- 先做3-5条测试，看数据反馈
- 不要追求完美，先跑通最小闭环

**4. 判断竞争格局**
- 红海还是蓝海？
- 你有什么信息差/资源优势？

**核心建议：** 小步快跑，先赚第一块钱，再优化放大。
